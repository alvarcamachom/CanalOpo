# CanalOpo

Repositorio de almacenamiento y organización de material de estudio, resúmenes y exámenes para la preparación de la oposición de administrativo del Canal.

## Contenido

El repositorio se divide principalmente en:

*   **Material de Estudio:** Documentos de teoría y resúmenes (como resúmenes de legislación y tarifas).
*   **Exámenes y Problemas:** Tests

## Acceso al Material

Todo el material compatible (archivos HTML y PDF) se puede visualizar online a través de GitHub Pages sin necesidad de descargarlo:

[Acceso web a CanalOpo](https://alvarcamachom.github.io/CanalOpo/)
